## Rumus Inti yang Sudah Disepakati

### HPP

- `effective_output_qty = output_qty * (1 - loss_percent/100)`
- `material_cost_i = qty_i * (1 + waste_percent_i/100) * unit_cost_i`
- `total_material_cost = SUM(material_cost_i)`
- `total_additional_cost = SUM(per_batch + per_unit * effective_output_qty)`
- `total_batch_cost = total_material_cost + total_additional_cost`
- `hpp_per_unit = total_batch_cost / effective_output_qty`

### Aset Persediaan

- `asset_value_item = qty_on_hand * avg_cost_per_unit`
- `total_inventory_asset = SUM(asset_value_item)` untuk item bahan baku/packaging
- Moving average saat stok masuk:
  - `q_new = q_old + q_in`
  - `v_new = v_old + (q_in * c_in)`
  - `avg_new = v_new / q_new`
